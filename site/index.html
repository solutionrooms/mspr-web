MS Paint Racers

A faithful, bit-exact TypeScript port of the 2010 Flash racing game.

The original is an ActionScript 3 / Flash game built on Box2DFlash 2.0.2 with a software sprite compositor and a procedural pseudo-3D road. This project rebuilds it in modern TypeScript — engine, game framework, and renderer — while reproducing the original physics exactly, verified mechanically against the shipped bytecode running under Ruffle.

The Prime Directive: the physics engine must match the original to the bit (hex16(ours) === hex16(original)) on every simulation step. A racing line, a ghost replay, and a medal time all desync if a single float drifts — so "close" is treated as a failure.

⚙️ Engine

Box2DFlash 2.0.2 ported to TypeScript line-by-line, gated bit-exact against golden traces.

🎨 Renderer

A WebGL2 compositor reproducing the software layer/blend/ColorTransform path + the road.

🕹️ Game

The GameObj/Levels framework, rebound to the bit-exact engine, driven by a fixed 2×(1/80) loop.
